Written in C++, Visual Studio 


Uses plaintext.txt for original text
Uses ciphertext.txt for encrypted text
Uses decrypted.txt for decrypted text


Key input for encrypt and decrypt modes accept 0-25, can accept higher numbers but uses modules to make them correct values.

Inputs to encrypt are in lowercase, outputs to uppercase.

Cryptanalysis operates under assumption that E is most frequent letter of unencrypted text.
